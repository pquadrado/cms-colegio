Informações

  • Adicionar Novo Evento
    1. Na barra de navegação lateral, clique em Adicionar Evento.
    2. Preencha o formulário, incluindo a data (no formato Dia-Mês-Ano) e o horário (formato Hora:Minutos).
    3. Uma imagem é obrigatória para criar o evento.
    4. A primeira imagem enviada será destacada como a principal do evento.
  • Visualizar e Remover Eventos
    1. Na barra de navegação lateral, clique em Eventos.
    2. Você verá todos os eventos já criados, incluindo detalhes como data e horário.
    3. Para remover um evento, clique no botão de Deletar ao lado do evento.
    4. Uma janela de confirmação será exibida para confirmar a exclusão ou cancelar a ação.
